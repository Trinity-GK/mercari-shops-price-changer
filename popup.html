Welcome to Mercari Shops Price Automation

How It Works

  • Automatically adjusts prices for all your shop products
  • Excludes products with pending shipments (発送待ち)
  • Restores original prices after a set time or when order threshold is reached
  • Runs in the background without interfering with your browsing

Requirements

  • You must be logged into mercari-shops.com in this browser
  • The extension uses your browser session (no API tokens needed)
  • Make sure you're logged in before starting automation
You can find your Shop ID in the URL when viewing your shop on mercari-shops.com

This setup will only appear once. Your Shop ID will be saved for future use.
